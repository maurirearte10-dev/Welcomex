# WelcomeX

Sistema de gestion de eventos con validacion de licencias.

## Descargar

Ve a la seccion [Releases](../../releases) para descargar la ultima version.

## Requisitos

- Windows 10/11
- Conexion a internet (para activar licencia)

## Instalacion

1. Descarga WelcomeX.exe desde Releases
2. Ejecuta el archivo
3. Ingresa tu clave de licencia
4. Listo!
